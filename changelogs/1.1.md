# 1.1.0
#### Including in this release:
- Moved `/checkperm` and `/listperms` (with `/listpermissions` alias) utility commands from DevTools into the core.
- Added default permission description translation keys and registered checkperm/listperms permissions under the Operator root.
- Added translations in English and Spanish for core permission command messages.
- Added support for case-insensitive command matching and execution (e.g. `/GAMEMODE CREATIVE` or `/gAmeMoDe CreaTIVE` correctly triggers `/gamemode creative`).
- Added the `settings.default-disable-vibrant-visuals` configuration option to `pocketmine.yml` to define whether shaders (vibrant visuals) are disabled by default server-wide, fully customizable on a per-player basis via `PlayerResourcePackOfferEvent`.
