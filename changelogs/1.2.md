# 1.1.2
#### Including in this release:
- Implemented non-ticking chunk unloading optimization from server memory (reducing active memory footprint of loaded chunks by up to 90%+ when render distance is higher than simulation distance).
- Implemented command overloading system (`OverloadedCommand`) to allow type-safe closures and parameter bindings via PHP 8 attributes.
- Migrated 40+ vanilla default commands to use the new overloaded command execution model.
- Added native command autocompletion mappings in `NetworkSession` for Bedrock client compatibility.
- Integrated canonical item autocomplete and icon rendering for the `/give` command using the Bedrock item type dictionary.
